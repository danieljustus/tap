# danieljustus/tap

A [Homebrew tap](https://docs.brew.sh/Taps) for Symaira tools.

## Usage

```bash
brew tap danieljustus/tap
brew install symmeet
```

Or install directly from the tap:

```bash
brew install danieljustus/tap/symmeet
```

## Available formulae

- `symmeet` — Local-first, standalone meeting-artifact tool for macOS 15+ (Apple Silicon).
